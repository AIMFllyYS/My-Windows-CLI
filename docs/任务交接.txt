  已实现功能

  命令行选项

  coding                 # 完整输出
  coding --short        # 简短输出
  coding --paths        # 仅项目路径
  coding --apps         # 仅应用启动
  coding --gh           # GitHub 账号 + Issues
  coding --gh-accounts  # 仅 GitHub 账号
  coding --gh-switch   # ⭐ 交互式账号切换（箭头键选择）
  coding --gh-issues   # 仅 Issues
  coding --cli <tool>  # CLI 参考 (cc/kiro/codex/gemini/cursor/all)
  coding --cli cc --task "任务"  # 带任务描述
  coding --chat        # AI 对话模式
  coding --ai          # AI 对话（alias）

  GitHub 模块

  - 解析 gh auth status 输出获取所有已登录账号
  - 支持账号切换 gh auth switch --user <username>
  - 交互式选择器（selector.ts 可复用）
  - Issues 查询（最近7天，按仓库分组）

  AI Chat 模块

  - DeepSeek API (deepseek-chat)
  - Read-only 模式（禁止编辑文件）
  - 工具：ls, dir, Read, Grep, WebSearch（模拟）
  - 多轮对话
  - API Key 存储在 .env
